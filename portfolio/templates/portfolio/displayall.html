About HoxNox

HoxNox is a simple portfolio generator created by Anupam Dagar. Just enter the required deatils and get you portfolio generated.The project is open source, pull requests are welcome. View Source

HoxNox features minimal and flat design.There is no place for spam and all it takes is 5 minutes to get your portfolio ready.

HoxNox includes full HTML, CSS, and custom JavaScript files along with LESS files for easy customization and Python's Django Framework as it's backend for fast and secure experience.
