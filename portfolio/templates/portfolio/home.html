About HoxNox

HoxNox is a simple portfolio generator created by Anupam Dagar. Just enter the required details and get your portfolio generated.The project is open source and pull requests are welcome. View Source

HoxNox features minimal and flat design. All it takes is just 5 minutes to get your portfolio ready.

It's reproducible which means the generated portfolio link can be shared on various platforms such as Facebook, LinkedIn, Twitter or anywhere else.

HoxNox includes full HTML, CSS, and custom JavaScript files for easy customization and Python's Django Framework as its backend for fast and secure experience.
